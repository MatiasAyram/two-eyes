# 🎬 two-eyes

Plataforma de streaming de video de alto rendimiento construida con tecnologías modernas. Permite a los usuarios explorar catálogos de contenido, visualizar tráilers, gestionar listas de favoritos y disfrutar de una experiencia de reproducción fluida y adaptativa.

# 🧑🏻‍💻 Tecnologías usadas

## Frontend

React, Next.js, TypeScript and Tailwind CSS

# Estado & Datos

TanStack Query (React Query) o Redux Toolkit.

# Backend

Node.js con NestJS.

# Base de Datos

PostgreSQL con Type ORM.

# Autenticación

JWT

# Almacenamiento de Video

AWS S3 / Cloudinary / Mux.
